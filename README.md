# Front starter pack (suggestion)



### Getting Started

- Clone o repositório

- Instale os pacotes
```
> npm install
```

- Exporte o css (escrito em scss)
```
> npm run build:css
```

- Inicie a aplicação

```
> npm start
```
 
- No desenvolvimento, levante o "watch" para observar todas as alterações de css e exportar na pasta "public/css"

```
> npm run watch:css
```
